Brain-Computer Interface Continuous Decoding EEG

Improving Continuous Grasp Force Decoding from EEG with Time Frequency Regressors and Premotor Parietal Network Integration

Parth G. Dangi, Yogesh Kumar Meena ¬∑ SMC 2025

A brain-computer interface system that decodes kinetic parameters of movement like force from brain activity recorded by EEG using simple regressors. This paper introduces EEGForceMap, a preprocessing and feature extraction pipeline to isolate brain activity encoding movement planning in premotor-parietal region and provides features representing planning of kinetic parameters as inputs to the regressors.
